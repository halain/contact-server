## Description

Contact List  API with NestJS and Prisma ORM

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Seeding DB

```bash
 # seed db 
$npx prisma db seed --preview-feature
```

## Test

```bash
# unit tests
$ npm run test

```
